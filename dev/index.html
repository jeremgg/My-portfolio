I intégration web

En collaboration avec les graphistes et les webdesigners, je développe un code optimisé selon les normes de la W3C pour un rendu accessible, propre et facilement maintenable.

Mes outils et technologies : HTML 5, CSS 3, Sass, javascript. Les frameworks et librairies que j'utilise le plus souvent sont : Bootstrap, Normalize, animatecss, jquery. Concernant l'editeur de code, je connais Bracket, Sublime Text mais j'ai une petite préférence pour Atom. Et pour finir j'utilise Gulp comme automatiseur de taches diverses pour permettre un gain de temps en production.

L'intégration des sites web que je développe offre une navigation la plus agréable possible en proposant une intégration responsive en mobile first. Ce qui permet d'avoir la meilleur experience possible quelque soit le type de média utilisé (smartphone, tablettes, ordinateur portable ou de bureau).

D développement web

En collaboration avec les autres développeurs front-end et back-end, je développe la partie back-end en PHP combiné à la base de données MySql pour par exemple gérer un système d'administration, un blog, un système de newsletters...
J'ai appris ce langage par l'intermédiaire de site web spécialisées ainsi que lors de ma formation de chef de projet multimédia à Digital Campus de bordeaux.
J'utilise également le gestionnaire de dépendance Composer pour installer les librairies dont j'ai besoin sur mes projets.

Concernant la programmation orienté objet avec PHP (POO), je met à jours mes connaissances par l'intermédiaire de cours et tutoriaux divers disponible en libre accès sur le web comme les sites openclassroom, codeAcademy ou encore Graphikart. La prochaine étape est de bien maitriser le framework Synfony.

Pratiquant une veille technologique régulièrement, je me met régulièrement à jour concernant les nouveaux outils, frameworks orientés Back-end.

W CMS Wordpress

En collaboration avec les webdesigners et les développeur front et back, je développe des sites internet et des blogs avec le CMS Wordpress.
Les pricipales extentions que j'utilise le plus souvent sont : Custom Post Type UI, Advanced custom fields, contact form 7, WP-PageNavi, révolution slider, Simple Custom CSS

Jai appris Wordpress par l'intermédiaire de ma formation de Chef de projet multimédia à Digital Campus de Bordeaux. J'ai pu mettre en application mes aquis lors de projets réels durant la formation.

Pratiquant une veille technologique régulière, je fais en sorte de bien connaitre les plugins ou les thèmes recommandés qui permettent aux clients de pouvoir naviguer dans le back-end de façon optimal et le plus simplifié possible.

Créer un site internet ou un blog via le CMS le plus répandu du web "Wordpress" pour permettre aux non initiés de gérer le contenu de leurs site simplement.
